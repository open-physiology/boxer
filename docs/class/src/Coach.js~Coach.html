    public registerArtefactEvent(events: Array<string>) source

    Let the coach know that it should start listening to specific types of DOM events on artefacts, and to provide a way to subscribe to them. It uses jQuery's delegation feature, so it is relatively efficient, and it normalizes event objects and supplies them with additional quick-access data, like the mouse event coordinates as a Point2D.

    Params:

    NameTypeAttributeDescription
    events Array<string>

    a set of DOM event names, e.g., 'click', 'mouseover'
