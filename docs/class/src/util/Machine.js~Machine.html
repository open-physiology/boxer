    public extend(descFn: Function, runCondition: function(): boolean): * source

    Extend this machine with additional states, behavior and/or transitions.

    Params:

    NameTypeAttributeDescription
    descFn Function

    a function that should return an object mapping state-names to functions that may (asynchronously, conditionally) initiate new state transitions in turn. It is passed an object with the enterState and subscribeDuringState. Use the former to enter new states (can also be bound to an observable). Bind the latter to an Observable instead of using .subscribe from within a machine state, so it is automatically unsubscribed at a state- transition.
    runCondition function(): boolean
    • optional

    a function that should dynamically indicate whether the machine is currently active; if omitted, assumes the machine is always active
